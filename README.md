Description
===========

This is a simple bash script to install XMind - Mindmapping Software on Linux. It is especially designed for non Debian based Distributions like Fedora.

Steps:
* Download XMind.deb
* Extract and Copy
* Update Cache

Usage
-----

Run the script with root privileges:

sudo ./install.sh
